# DIY Internship Task 3 - Website Performance Testing

## Project Title
Test Website Performance Under Load

## Description
This task focuses on measuring the website's performance, response time, and stability under varying traffic loads using JMeter.

### Test Objectives
- Measure page load time under normal and high traffic.
- Observe website stability under simulated user load.
- Record and evaluate key performance metrics such as latency and throughput.

### Files Included
1. Test_Case_Performance.xlsx - Contains detailed performance test cases.
2. Test_Report_Performance.xlsx - Contains test execution results (all tests passed).

### Tools Used
- Apache JMeter
- Manual Observation
- Microsoft Excel
- GitHub for documentation
