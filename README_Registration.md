# DIY Internship Task 2 - Manual Testing on User Registration Page

## Project Title
Validate Usability of User Registration

## Description
This task focuses on testing the ease of use and validation process for the user registration page of an e-commerce website.

### Test Objectives
- Verify that the registration process is user-friendly.
- Validate input fields for correct and incorrect data.
- Check for appropriate validation messages.

### Files Included
1. Test_Case_Registration.xlsx - Contains detailed test cases.
2. Test_Report_Registration.xlsx - Contains test execution results (all tests passed).

### Tools Used
- Manual Testing
- Microsoft Excel
- GitHub for documentation
